# How to enabled PWA in vuepress

TIP

We need some plugins to enabled PWA

# Install

yarn add -D @vuepress/plugin-pwa
# OR npm install -D @vuepress/plugin-pwa

# Use

module.exports = {
  plugins: ['@vuepress/pwa']
}

To make your website supported PWA completely.You should add Manifest and icons

For more information https://developer.mozilla.org/en-US/docs/Web/Manifest
